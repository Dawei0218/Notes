# 快速移动

### 快速移动

```
以下在普通模式下

hjkl

word是非空白分割的单词，WORD是空白分割的单词

w/W 下一个word/WORD开头
e/E 下一个word/WORD结尾
b/B 回到上一个word/WORD开头

f{char} 搜索字符跳转 ;移动到下一个 ,移动到上一个
t{char} 搜索字符的前一个字符
F{char} 反后来搜索

0 行首第一个字符
^ 第一个非空白字符
$ 行尾
g_ 行尾非空白字符

gg/G 文件开头/文件结尾 <C> + o 快速返回
H/M/L 屏幕开头 中间 结尾

<C> + u, <C> + f 上下翻页
zz 把屏幕置为中间
```
