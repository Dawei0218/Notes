# 多文件操作

### 打开其它文件

```
:e a.txt
```

------

### buff

```
打开一个文件就会有一个buff
:ls 列出所有buff
:bpre  :bnext  :bfirst  :blast  :b buffer_name
```

------

### 窗口

```
可视化分割区域
一个缓冲区可以被分割成多个窗口，每个窗口也可以打开不同的缓冲区
<C + w> + s 水平分割 或者:sp
<C + w> + v 垂直分割 或者:vs

切换窗口
<C + w>w 在窗口间循环切换
<C + w>h 切换到左边窗口
<C + w>j
<C + w>k
<C + w>l
```
