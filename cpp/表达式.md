# 表达式

### 优先级和结合律

```c
// 高优先级运算符的的运算对象要比低优先级运算符运算符对更紧密集合在一起
// 如果优先级相同，则其组合规律由结合律决定
// 括号无视优先级与结合律
3 + 4 * 5 = 23;  // 不等于35
20 - 15 - 3 = 2; // 不等于8
```
