# 控制语句

### if-else

```c
if(true)
    语句1;
else
    语句2;

expr1 ? expr2 : expr3;

// else会与最近一个没有else匹配的if匹配
if(true)
    if(true)
        语句1;
else
    语句2;

if(true)
    语句1;
else if(true)
    语句2;
else
    语句3;
```

------

### switch

```c
switch (a) {
    case 1: 语句1; break;
    case 2: 语句2; break;
    default: 语句3; break;
}
```

------

### while

```c
while (true) {}
```

------

### for

```c
for (表达式1; 表达式2; 表达式3) {}
```

------

### do-while

```c
do {} while (true)
```

------

### goto

```c
goto err;

err:
    错误处理情况;
```
