# 运算符

###  运算符优先级

| ***运算符***                  | ***结合性*** |
| ----------------------------- | ------------ |
| () [] -> .                    | 自左向右     |
| ! ~ ++ -- - (type) * & sizeof | 自右向左     |
| * / %                         | 自左向右     |
| + -                           | 自左向右     |
| << >>                         | 自左向右     |
| < <= > >=                     | 自左向右     |
| == !=                         | 自左向右     |
| &                             | 自左向右     |
| ^                             | 自左向右     |
| \|                            | 自左向右     |
| &&                            | 自左向右     |
| \|\|                          | 自左向右     |
| ?:                            | 自右向左     |
| = += -= *= /= %= &=           | 自右向左     |
| ,                             | 自左向右     |

-----

### 自增自减

```c
n++;
++n;
x = n++; // 先赋值在加
x = ++n; // 先加在赋值
```
