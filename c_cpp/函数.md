# 函数

### 函数

```c
// void是ANSI C标准，一些编译器允许第二种写法，但是所有标准都未认可
// c++括号内不需要void
int main(void) {}
void main() {}

// 函数原型，任何函数在使用函数前都要声明该函数原型
void func(void);

// 带形式参数的函数
void func(int num);

// 只有在函数定义和函数原型的时候才可以用ar[]代替*ar
// 对于数组只能传递指针，如果不想被修改可以加上const限定符
int sum(int ar[], int num);

// a+1 = 该地址加上16个字节，如果两对括号是空，编译器不知道怎么处理
int sum(int a[][]); // 错误
int sum(int a[][4]); // 正确
```

### 函数指针

```c
int sum(int a, int b)
{
    return a + b;
}

int main(void)
{
    int (*p)(int, int) = sum;
    
    (*p)(1, 2);
    
    return 0;
}
```
