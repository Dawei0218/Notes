# 常用编译命令

### 预处理命令

```
gcc -E hello.c -o hello.i
```

------

### 编译

```
gcc -S hello.i -o hello.s
```

------

汇编

```
as hello.s -o hello.o
gcc -c hello.s -o hello.o
```

------

### 链接

```
ld xxx
```

------

### 自定义段

```c
// 可以放到指定的段里
__attribute__((section("Foo"))) global = 42;
```

------

### 查看ELF Header

```
readelf -h test.o
```
